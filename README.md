# FinancePro

## Overview
FinancePro is a comprehensive financial management tool designed to help users track their spending, create budgets, and gain insights into their financial habits. The application provides a user-friendly interface and powerful features to manage personal finances effectively.

## Features
- **Expense Tracking:** Easily track and categorize your expenses.
- **Budgeting Tools:** Set monthly budgets and track your spending against them.
- **Financial Insights:** Gain insights through charts and reports on your spending habits.
- **User Authentication:** Secure sign-up and login process.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/lucasfonsec/FinancePro.git
   ```
2. Navigate into the project directory:
   ```bash
   cd FinancePro
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```

Visit `http://localhost:3000` in your web browser to access the app.

## Technologies Used
- Node.js
- Express
- MongoDB
- React
